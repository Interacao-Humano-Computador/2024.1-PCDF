    Introdução

    Escolhemos essas ferramentas para serem utilizadas durante o processo de desenvolvimento do nosso projeto de Interação Humano Computador.

    Objetivo

    Esse artefato te como objetivo documentar as ferramentas

    Metodologia

    Com o decorrer do projeto fomos ultilizando algumas ferramentas e depois fomos colocando no artefato

    Ferramentas Utilizadas

    Primeira Tabela, informações de todas ferramentas e suas utilidades para o projeto

    Número Logo Ferramenta Finalidade
    1. Whatsapp Usamos o Whatsapp para facilitar a comunicação entre os membros do grupo.
    2. Teams Utilizamos o Teams para realizar nossas reuniões durante a semana e fazer as gravações.
    3. Youtube Usamos o Youtube para colocar as gravações do projeto.
    4. VsCode IDE utilizada para o desenvolvimento do projeto.
    5. GitHub Utilizada para hospedar o repositório e a git page.
    6. Exel Utilizado para fazer o Heatmap.
    7. Forms Utilizado para fazer o Questionário.
    8. Figma Usado para prototipo e Storyboard
    9. Miro Utilizado para trocar ideias
    10. Storyboard That Utilizado para fazer o Storyboard

    Detalhamento das ferramentas utilizadas durante o projeto.

    (Fonte: SILVEIRA, Raissa. 2024).
